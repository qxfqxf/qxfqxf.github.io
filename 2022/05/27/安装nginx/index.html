y要先安装所需插件

yum的话应该是需要切换到root才可以使用

1：安装gcc 这个一般都是自带都有 查看一些 gcc -v 如果没有 可以yum -y install gcc

2: pcre, prce-devel安装 yum install -y pcre pcre-devel

3:安装 zlib yum install -y zlib zlib-devel

4：安装openssl yum install -y openssl openssl-devel

5: 下载Nginx wget http://nginx.ort/download/nginx-1.16.1.tar.gz

这里会下载一个压缩文件

6:解压nginx压缩文件 tar -zxvf nginx-1.16.1.tar.gz 这里应该是要注意下目录的

7:写如配置文件，为编译安装做准备 (先试一下make，可不可以编译，不可以的话就写配置文件)

8:编译安装

make

make install

9:启动 /usr/local/nginx/sbin/ngiinx

如果不能正确启动 ，那就是报下面的错误，

nginx: [emerg] mkdir() “/var/temp/nginx/client” failed (2: No such file or directory)

意思是少一个文件夹，以下方法就可以解决

[root@localhost sbin]# sudo mkdir -p /var/temp/nginx

如果运行成功 但不显示的话，可以关闭防火强，systemctl stop firewalld 开启就是start

这样的话就成功在centos中安装nginx了，并成功运行， 下一步是把vue项目部署到centos上
