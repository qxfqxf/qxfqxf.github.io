流程

dir 查询文件目录
hexo serve 启动
hexo n “我的第一篇文章”
hexo clean清除了你之前生成的东西，也可以不加。
hexo generate 顾名思义，生成静态文章，可以用 hexo g缩写
hexo deploy 部署文章，可以用hexo d缩写

添加标签： 在需要添加标签的文章里在data后加

categories：

​ - Python

部署到github 需要下载hexo-deployer-git
需要到_config.yml下的Deployment去配置type=”git”和git地址
然后就可以进行部署 hexo d
